Bienvenido(a) a mi primera aplicación de tipo PWA

¿Qué expone esta app?

Esta app explica que es una PWA y unos simples pasos de introducción para crear una.

¿Qué es una PWA?

Primero la definición de sus letras. PWA significa Progressive Web App (traducido al castellano, App Web Progresiva). Las apps de tipo PWA se desarrollan con tecnologías web, por tanto se pueden usar lenguajes HTML, CSS y JS para crear nuestra app.

Una PWA es una página web la cual contiene unos códigos extra que ordena al navegador del usuario invitarle a añadir esta página al escritorio, cuando el usuario realice esto, se creará un icono en el menú simulando ser una app normal.

La app PWA añadida por el usuario será como una app nativa, tendrá interfaces, iconos, etc, simulando ser una app normal.

¿Cómo añadir una PWA a mi smartphone?

El usuario debe dirigirse a la página web usando un navegador moderno y que permita instalar PWAs. Cuando el usuario esté en dicha página, el navegador solicitará permiso para añadir la nueva app, el usuario debe aceptar añadir la app al menú, y listo, la app fue instalada.

¿Cómo yo puedo crear mi propia app PWA?

Para craer una app PWA, principalmente debes desarrollar un proyecto web, a este proyecto web le agregas unos códigos complementarios como se explica a continuación, y listo, terminarás tu app PWA.

¿Cuáles son los pasos para crear mi PWA?

En simples pasos:

Paso 1. Crea el Manifest.json. En este archivo se declara el nombre, los iconos, la orientación de la app, etc, cosas acerca de nuestra app. Para más información visita la página oficial la cual continene todos los detalles (https://w3c.github.io/manifest/).

Paso 2. Crea el service-worker.js. Es crear un JS especial en donde se ejecuta la instalación de la PWA y se guarda en caché los recursos y utilidades de la app.

Paso 3. Registra el service-worker.js desde un JS. Es añadir unas pequeñas lineas de código que permiten saber si el navegador es compatible con el Service Worker, si es así, se procede a registar nuestro archivo service-worker.js.

Nuestro archivo service-worker.js puede ir en cualquier lugar, aunque normalmente se elige ubicarlo paralelamente al index.html. El archivo service-worker.js no se declara usando la etiqueta script aunque este sea un script; este archivo se mantiene oculto y no aparece en ninguna parte del index.html
